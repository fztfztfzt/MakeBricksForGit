MakeBricksForGit
================

a game like eliminate the stars
This is make by MFC of vs2010

1. The game can be set to the width, height, color variety and sound effects
2. The game can be saved at any time, with the fast preservation and save to file, exit will automatically save the game
3. The game is a classic model, classic mode is imitation phone model, classic mode is the highest and level Settings
4. The classic mode do not change the width of the game, height and color types
5. The game's sound effects and background images under the res folders, games that use the font for "classical interest body Jane", if no font file can be installed under the res folder
